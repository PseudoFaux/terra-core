# Terra Form Input

The Input component is a form input for entering single line text.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-form-input`

## Usage

```jsx
import React from 'react';
import Input from 'terra-form-input';

<Input
  name="medication"
  value="Aspirin"
  required
 />
```
